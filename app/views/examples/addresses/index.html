Address lookup

Patterns for helping users find an address from a list. Test each example with this address: 7 Pegasus Place, St Albans AL3 5QT.

The patterns

1. Auto-suggest box

This pattern lets users start typing their address and then uses that to return entries from the full register. The main challenge is how quickly you can return a useful set of addresses.

2. Postcode plus auto-suggest box

You can narrow the search space of pattern 1 considerably by having users enter their postcode first. This won't help people who don't know the postcode though, so make it optional.

3. Postcode plus embedded select box

An alternative to pattern 2 is to present the results in a select box, so the user only has to type their postcode.

Some users really struggle to scroll through long select boxes. This custom version addresses that issue by opening out in full and moving the content below it down, so as not to obscure it.
